# Novaflow — Edelmann Lab demo

A tailored workspace preview for the Edelmann Lab (Department of Cell Biology,
Albert Einstein College of Medicine).

The walk-through shows the everyday loop: load colon RNA-seq counts, run a
DESeq2 + GSEA pipeline, surface MMR / Wnt hits, and cross-reference against
the lab's organoid scRNA signature — all from chat.

- Frontend: **nfx** (Novaflow's bioinformatics CLI / UI layer)
- Backend: **Feynman** research agents

Open `index.html` directly in a browser, or view the live deploy on GitHub Pages.
